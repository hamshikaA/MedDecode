# MED-DECODE  
### AI Medical Report Translator
MED-DECODE is an AI-powered web application that helps users understand complex medical reports by converting them into **simple, easy-to-understand English**.
This application does **NOT** provide diagnosis or medical advice.
## Features
- Upload **medical report images or PDFs**
- Extracts text using **OCR (Tesseract)**
- Explains **medical terms in simple language**
- No diagnosis or treatment suggestions
- Privacy-focused and ethical AI usage
## Tech Stack
- Python
- Flask
- Tesseract OCR
- Google Gemini API
- HTML / CSS
